# @pokemart/web

Ứng dụng Next.js của PokéMart.

## Chạy dev

```bash
pnpm dev
```

Mặc định web chạy tại **http://localhost:3040** (cổng cấu hình trong `package.json`, không dùng 3000).

API backend cần khớp với `NEXT_PUBLIC_API_URL` trong `.env.local` (mặc định `http://localhost:3041/v1`).

Xem thêm [README gốc monorepo](../../README.md).
